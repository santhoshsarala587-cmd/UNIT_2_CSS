# UNIT_2_CSS

Description:

This unit focuses on CSS (Cascading Style Sheets), which is used to style and design HTML web pages. It covers CSS syntax, selectors, colors, backgrounds, fonts, and text styling. It also includes advanced concepts like the box model, flexbox, grid layout, positioning, animations, transitions, and responsive design. This unit helps in improving the visual appearance and layout of websites and creating modern, responsive web designs.

Topics Covered:

CSS syntax and selectors

Colors, backgrounds, and typography

Box model (margin, padding, border)

Display and positioning

Flexbox layout

Grid layout

Responsive design (media queries)

Transitions and animations

CSS variables

Website project using HTML & CSS

Outcome:

By completing this unit, I learned how to design responsive and visually attractive websites using CSS.
